Snake Game

A classic Snake game built using HTML, CSS, and JavaScript.

Game Description
Snake is a popular arcade game where the player controls a snake that eats food pellets and grows in length. The game ends when the snake collides with the wall or its own body.

Features
- Simple and intuitive gameplay
- Score tracking
- Increasing difficulty level
- Smooth animation

How to Play
1. Use the arrow keys to control the snake's direction.
2. Eat the food pellets to grow the snake's length.
3. Avoid colliding with the wall or the snake's own body.
4. Try to beat your high score!

Installation
1. Clone the repository: `git clone https://github.com/whoops-1/Snake-game.git`
2. Open `index.html` in your web browser.

Code Structure
- `index.html`: The main game file.
- `styles.css`: The game's CSS styles.
- `script.js`: The game's JavaScript code.
